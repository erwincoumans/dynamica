Release Notes C NVIDIA PhysX Plugin for 3DS Max

Version 1.0.6 Beta

Oct. 10, 2008

 

 

Introduction

This tool provides interfaces for a user to create NVIDIA PhysX objects inside 3DS Max.

Because the release depends on PhysX system software, user is recommended to install the 08.06.12 version of NVIDIA PhysX driver which can be downloaded at http://www.nvidia.com/object/physx_8.06.12_whql.html before using the Plugin.

Changes

・        Support CCD (Continuous Collision Detection) for better collision result with less penetration

・        Support changing the material of a rigidbody via script

Usage:

float    px.getDynamicFriction(INode *node);

float    px.getStaticFriction(INode *node);

float    px.getRestitution(INode *node);

 

int      px.setDynamicFriction(INode *node, float value);

int      px.setStaticFriction(INode *node, float value);

int      px.setRestitution(INode *node, float value);

 

・        Support updating rigidbody’s position and pose via script

Usage:

Point3            px.getGlobalPosition (INode *node);

int            px.setGlobalPosition (INode *node, Point3 &position);

Matrix3    px.getGlobalPose(INode *node);

int            px.setGlobalPose(INode *node, Matrix3& pose);

・        Support sub-simulation steps for accurate collision via script

Usage:

      int             px.setSubSimSteps(int numSteps);

・        Fix a bug related to rigidbody proxy

Known Issues

  • Occasionally, the user may have to press ‘reset’ several times to make cloth actors go back to their original state.
